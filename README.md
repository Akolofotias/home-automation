## Precompiling client-side application

    -   Install node.js

    -   Install npm-packages

        $ cd /path/to/automationFolder
        $ npm install

    -   Compile front-end to /path/to/automationFolder/htdocs/dist.

        $ grunt

## Running api-tests

    - Install node.js and npm-packages

    - Running tests

        $ grunt mocha
